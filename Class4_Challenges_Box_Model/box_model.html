CSS Box Model

Elements can be divided into two dispaly systems: Inline and Block

Inline

  • Always starts on a new line
  • Take up full width available
        <span> Hello </span>
        <span> World </span>

Common Inline Elements

  • Spans <span>
  • Images <img>
  • Anchors <a>

Block

  • Does not start on a new line
  • Only takes up as much width as needed

Common Block Elements

  • Paragraphs <p>
  • Headers <h1> through <h6>
  • Divisions <div>
  • Lists and list items <ol> , <ul>, and <li>
  • Forms <form>
        <div> Hello </div>
        <div> World </div>
